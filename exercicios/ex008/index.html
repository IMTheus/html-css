principais formatações

Negrito/Destaque

Nesta frase vou colocar em NEGRITO usando a tag B

Essa tag é não semântica --significa apenas forma não tem significado--

Nesta frase vou colocar em NEGRITO usando a tag STRONG

Essa tag é semântica--significa que tem significado, não apenas forma--

Italico/Ênfase

Nesta frase temos um termo em italico usando a tag I

Essa tag é--Não semântica

Nesta frase temos um termo em italico usando a tag EM

Essa tag é semântica

OBS Sempre bom usar as tags semântica

Texto com Marcação

para criar um texto com marcação ,basta add a Tag MARK

Para mudar a cor do marcador,basta add a tag STYLE - BACKGROUND COLOR e selecionar a cor ( dentro da tag MARK),mas isso ja está em CSS3

Texto grande e Texto pequeno

o texto grande é usado pela tag BIG,mas ja é uma tag Inesistente,porém funcional

o texto pequeno é usado pela tag SMALL e ainda é uma tag funcional

Texto deletado

Podemos marcar um texto como excluido para marcar que ele deve ser lido,mas nao considerado(TAG usada para texto excluido é DEL)

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e marcar que ele foi add depois(TAG usada para texto inserido é INS)

Texto sobrescrito

Usado para inserir coisas do tipo X2+X3-3 (TAG é SUP )

Texto Subscrito

Usado para inserir coisas do tipo H2O (TAG é SUB)
