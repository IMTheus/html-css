para mostrar que o link já foi visistado e escolher a cor,basta add (A:VISITED{COLOR} para a cor)
para mostrar uma cor em cima do seu link após clicar,basta add (A:ACTIVE{COLOR} para a cor)
para mostrar algo antes ou depois do link de maneira personalizada basta add a tag (A::AFTER{CONTENT'PALAVRA'} AFTER= DEPOIS e CONTENT=CONTEUDO)
para add algo depois add (A::BEFORE{CONTENT'PALAVRA'} BEFORE=ANTES e CONTENT=CONTEUDO).
