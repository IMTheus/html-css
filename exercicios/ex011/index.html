Imagens dinamicas

Tente abri esse site em varios dispositivos ou aumentar e diminuir a miagem em seu navegador

Resumo

Para colocaruma imagem flexicivel em um site que se adapte em qualquer navegador,Primeiro coloque as medidas
300 é pqueno para celular
700 é médio para tablets
1000 é para o computador

Para add as imagens,basta add a TAG PICTURE e dentro dela add a TAG IMG selecionar as TECLAS CTRL-SPACE selecionar a imagem e no ALT add IMAGEM FLEXÍVEL em cima disso tudo add a TAG SOURCE-MEDIA-TYPE,dentro da MEDIA add MAX e o Tamanho da imagem, em SRCSET add o nome da imagem e em TYPE o formato dela EX(PNG)

Reproduzindo audios

Reproduzindo audio em sites HTML5

Maneiras de add um audio é add a TAG AUDIO e dentro de SRC add CTRL+SPACE e selecionar seu Audio e add depois das aspas e antes da TAG final AUDIO, CONTROLS AUTOPLAY
