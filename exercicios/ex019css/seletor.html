A tag ID é usada uma vez apenas, pode ser para qualquer coisa mais geralmente é o prmeiro titulo ela tem como objetivo separar das demais outras. digite ID dentro da tag que for usar.


EX: H1 qualquer coisa que colocar vai servir para todos os H1. Quando usar H1 ID "COLOQUE A PALAVRA PARA DESTAQUER APENAS ESSE H1 DENTRO DAS ASPAS" e esse h1 ira ficar isolado.

A tag CLASS pode ser usada quantas vezes quiser, ela tem o mesmo objetivo que a tag ID, mas pode ser usada quantas vezes quiser. Digite CLASS dentro da tag que for usar.


EX: H2 CLASS"BASICO" digitando essa tag no STYLE.CSS, apenas essa se destacara das outras. Caso tenha duvida olhe o ex 19.

A tag SPAN tem como objetivo selecionar palavras em meios de texto e dentro dessa tag use a tag CLASS para fazer uma marcação.
